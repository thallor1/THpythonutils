powderNXSPEtoMDHisto(q_slice, e_slice, file_arr=None, mt_arr=False, self_shield=1.0, eventnorm=True)[source]

Takes file array and turns it into MDhisto workspace (powder measurements only).

Parameters:

  • file_arr (list,str) – List of reduced NXSPE files to import, or a single file. Input to this function will default to the .filename attribute, but can be overwritten using this argument.

  • mt_arr (list,str) – Optional list of background measurements to subtract directly.

  • q_slice (list) – List in the form of [Qmin,Qmax,Qstep].

  • e_slice (list) – List in the form of [Emin,Emax,Estep]

  • self_shield (float) – Self shielding factor if MT array is specified for background subtraction.

  • eventnorm (bool) – Flag to normalize by number of monitor events, defaults to True.
