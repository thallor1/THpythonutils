thpyutils.neutron.magAnalysis.factorization module

thpyutils.neutron.magAnalysis.factorization.factorization(q_values, energies, intensities, errors, mag_ion=None, qe_limits=False, method='powell', fname='uncertainties.txt', fast_mode=False, overwrite_prev=False, g_factor=2.0, fix_Qcut=False, fix_Ecut=False)[source]
Parameters:

  • q_values – Input x-axis (or momentum transfer) values.

  • energies – Input y-axis (or energy transfer) values.

  • intensities – Input matrix of intensities

  • errors – Input matrix of error bars

  • mag_ion – Magnetic ion of sample, default is None.

  • qe_limits – Limits of the factorization region in format of [Qmin, Qmax, Emin, Emax]

  • method – Least-squares fitting method, default is Powell.

  • fname – Filename to save result to.

  • fast_mode – Flag to just assume a 30 percent error bar.

  • overwrite_prev – Flag to overwrite previously calculated errorbars

  • g_factor – G-factor of magnetic ion, default 2.

  • fix_Qcut – Flag to fix the factorization in Q to be the same as an averaged cut.

  • fix_Ecut – Flag to fix the factorization in E to be the same as an averaged cut.

Return result:

Array of [q, sq, sqerr, e, ge, geerr].

thpyutils.neutron.magAnalysis.getMagFF.getMagFF(q, mag_ion)[source]

Given a list of momentum transfer magnitudes, returns a list of equal size of the magnetic form factor squared or |F(Q)|^2. Wrapper for Mantid catalogued form factors.

Parameters:

  • q (np.ndarry) – Array of momentum transfers

  • mag_ion (str) – String containing ion and valence in MANTID notation, for example ‘Ir4’. See Mantid documentation for allowed values.

Return FF:

List of values by which data must be divided.
